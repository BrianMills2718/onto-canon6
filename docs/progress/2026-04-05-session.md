## Key Discovery

`extract-progressive` uses `sumo_plus.db` directly via `PredicateCanon` and `SUMOHierarchy` in its 3-pass pipeline — it does NOT use ontology packs. So the path to typed predicates is:
1. Run `extract-progressive` on Iran text  
2. The SUMO predicate mapping happens automatically in Pass 2
3. Submit + promote to `onto_canon6.db`

Building a vocabulary pack from `sumo_plus.db` is deferred — it's needed for profile-based regular extraction, not progressive extraction.
